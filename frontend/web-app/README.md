# StreetEats Live — Web App (React + Leaflet)

Live map of nearby street-food vendors. Pins are colour-coded by status
(green=open, amber=moving, grey=closed/unconfirmed, red=no status). Tap a pin
for vendor info and a "Here now ✓" crowd-confirm button.

## Run
```bash
npm install
npm run dev      # http://localhost:5173  (proxies /api to the services)
```
Falls back to demo data if the backend isn't running.
